# Asphalt Files

This is a reupload of Asphalt Files.
To save those who are interested the time of NOT downloading a 22GB Zip

## Included in the repo

This repository contains the essential compiled files and data:

- Compiled executables (.exe files)
- Training data
- Configuration files
- Documentation

## Removed junk

- Build files
- Development dependencies
- Temporary build artifacts
- Pycache
